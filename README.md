# 🚀 Explain My Plan

An AI-powered web app that converts messy ideas into **clear, structured, and actionable plans**.

---

## ✨ Features

- 🧠 AI-powered idea structuring (Gemini API)
- 📊 Dynamic Clarity Score (0–100)
- 📌 Goal, Method, Steps, Timeline breakdown
- ⚠️ Missing elements detection
- 🔁 Before vs After comparison
- 🎯 Actionable next steps
- ⚡ Clean and modern UI (Next.js + Tailwind)

---

## 🛠️ Tech Stack

- **Frontend:** Next.js, React, Tailwind CSS
- **Backend:** Next.js API Routes
- **AI:** Google Gemini API
- **Language:** TypeScript
